(taken by kawataso)

去る2009年3月15日，おびなたさん主催の巨大構造物撮影会に参加してきました．

巨大構造物撮影会 : ATND

ちょうど，はしゃぎ足りないタイミングだったこともあって，皆さんとの時間をとても楽しく過ごすことができました．ありがとうございます．また，「撮影会」という名目で8人もの人が集まり，そこで撮られた写真があとから Web 上で共有されると，同じく歩いた道で自分以外の人たちがどんな景色を見ていたのかが分かって大変面白いですね．

去年の終わり頃に今年，最も面白かったソーシャルアプリはFlickrだというエントリを書きましたが，今年も Flickr には感謝し続けることになりそうです．

写真

この日は900回ほどシャッターを切りました．

巨大構造物撮影会 – a set on Flickr
