Firefox 2.0とエクステンション

2006-11-08

Firefoxを2.0に更新してから数日が経ちました．1.5で利用していた機能はほぼすべて2.0でも利用できているので，今のところ快適にブラウジングできています．Firefox楽しい！Firefoxパーカーほしい！

導入中の拡張機能とカスタマイズ

Firefoxユーザの方たちがよくやっている「公開！アタシのお気に入りエクステンション！」を初めてやってみます．有名どころばかりであまり面白くありません．

All-in-One Gestures

すごく便利に使っているけれど，最近はキーボードショートカットに慣れてきたからそのうち使わなくなるかもしれません．

Tab Mix Plus

これ待ちで2.0への移行をためらっていました．ロケーションバー，ブックマーク，履歴などから開くページはすべて新規タブに開き，フォーカスはそのままにするのが好きです．

Greasemonkey

グリモン大好き！自分でもスクリプトをバリバリ書けるようになりたい！要javascript勉強．

FEBE

拡張機能をxpiとしてバックアップ．再セットアップ時に便利です．

CLEO

複数のxpiを1つのxpiに統合．上記のFEBEと組み合わせて使います．

functions for keyconfig，keyconfig

ブラウジング中は「Ctrl+j」「Ctrl+k」でタブの切り替え，「Ctrl+w」でタブを閉じる，「Ctrl+l」でカーソルをロケーションバーに移動，この辺りをよく使います

Web Developer，FireBug，IE Tab

開発用．最近はあまり出会わないけれど，Firefoxで上手く表示されないページを見るときにIE Tabを使ったりします．

Copy URL +

リンク作成支援．エントリ投稿時には必須です．

Favicon Picker 2

ブックマークツールバーのブックマークレットをアイコン表示にするために使っています．

dragdropupload

ファイルのアップロード時，D&Dでファイルのパス指定．そもそもすべてのブラウザで標準実装していてほしいくらいです．

MR Tech Link Wrapper

横に長いリンクをウィンドウ幅で折り返し．

MR Tech Local Install

Firefoxのクイックリスタート，Chromエディター，アドオン管理支援など様々な機能のパッケージ．

Resizable Form Fields

フォームの大きさを自由に変えられるように．ブログのコメント欄などを大きく使いたいときに便利です．

こんなこといいな

大好きなFirefoxだけど「もっと○○だったらなぁ…」と思うことはあります．もしかしたらボクが気が付いていないだけで実現されていることもありそうですが，調べても見つからなかったのでここに書いておきます．ちなみにWindows XP環境下での話です．

日本語入力ON時にもキーボードショートカットが有効だったらな

キーボードでGmailを操作するときや，ページ内クイック検索で「/」を使うときなど，日本語入力がONになっていると反応してくれないのは困りますね．英語圏に生まれたかったなぁと思ってしまう瞬間です．

ページ内検索のボックスをキーボードショートカットで表示/非表示切り替えできたらな

色々調べてみたのですが，未だに実現方法が分かりません．「Ctrl+f」で開いたページ内検索ボックスを非表示にするとき，皆さんはどうしていますか．All-in-One Gesturesの設定項目内にはありますが，functions for keyconfigの設定には見つけられません…．わざわざマウスカーソルを動かして「×」ボタンを押すのはあまりにも効率が悪い気がします．良い方法があったら教えてください．

(追記) Firefoxのキーコンフィグ : Ctrl+Fで検索バーの開閉 – cameraLadyにて解決策を示しています．
