MacBook Pro 転生の儀

2013-09-23

SSD を大容量のものに換装してマッコォエステンッのトァイムマッスィィインから魂を復元したんだけど、なんかデータは引き継がれつつ、システムやアプリケーションの設定なんかは妙に初期化されていたりして、記憶に障害が残った感じで、まぁそれはそれで仕方ないね、と思いながらひとつずつ丁寧に「気付いたらそう設定してずっと使っていた設定」を取り戻していった。気付いたら設定する、ってのを続けている。そのメモ。

  • KeyRemap4MacBook
    • 「左右の Command キーをかな/英数に割り当て」
  • システム環境設定 - キーボード
    • 「Ctrl + (n)」で「デスクトップ(n)へ」
    • 「Ctrl + Space」で Spotlight を開かないように (for Emacs)
  • イメージキャプチャ
    • この iPhone 接続時に起動するアプリケーションに「割り当てなし」(iPhoto が起動しないように)
  • Mailplane3
    • ライセンスキーを入力
  • TotalFinder
    • 「Ctrl+Command+T」でバイザーを起動するように
  • iTerm
    • profile を新たに作成

あと、このタイミングに乗って Alfred を導入した。なんも問題なく使えている様子。
